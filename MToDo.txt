[パラメーター]
・サイズ適応範囲をBlockParamに設定して適応サイズに当てはめていく
・DTにParameterStructを追加して設定していきアセット管理をする

[DataTable]
・アセットスケールを1以下のランダムにして変化をつける

○DTから値を取得し反映させる処理を実装する
・パラメーターごとにSprit + find + atofで値を抽出

・ソース
string ParamStrArray[] = split(StringTest);
string ParamStr = ParamStrArray[0];
int StrStartIndex = find(ParamStr, "X") + 2;
int StrEndIndex = StrStartIndex + 8;
string ValueStr = ParamStr[StrStartIndex:StrEndIndex];
@TransformX = atof(ValueStr);

[BaseBlock]
・サイズに変化を持たせる

○Greebleロジックの設計
 
○DTとBaseBlockを検証する
・※BaseBlockのサイズに変化を持たせて様々な適応サイズのDTを用意しサイズ、種類ともに変化を持たせる

[BaseSurface]
○BaseSurface,BaseBlockの生成ロジック
・Voronoiで角度、サイズに変化があるSurfaceを生成するロジックを設計する
